Responsive image learning (and art-direction)

View this page in various viewport widths to see the images below change with art-direction. The web browser will determine which image to load.

Usage of the HTML5 image-src-set and the picture -tag.

Different cropped versions of the same image. Displaying at various viewport width to keep the focus in the image to a certain subject. And keeping the file-size, and therefor, the bandwidth low. Good for performance.

HTML demo's

HTML img-tag with srcset & sizes attributes

<img
  src="/img/obama-large.jpg">
  width="1080" height="726"
  alt="Barack Obama presenting Purple Hearts"
  srcset="/img/obama-large.jpg 1080w, /img/obama-medium.jpg 768w, /img/obama-small.jpg 480w"
  sizes="(min-width: 480px) 100vw, 100vw">

HTML picture-tag with source-elements

<picture>
  <source srcset="/img/obama-small.jpg" media="(max-width: 800px)">
  <source srcset="/img/obama-medium.jpg" media="(max-width: 1200px)">
  <img src="/img/obama-large.jpg" width="1080" height="726" alt="Barack Obama presenting Purple Hearts">
